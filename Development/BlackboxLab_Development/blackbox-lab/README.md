# Blackbox Lab v1.0

**Engineering Intelligence for RotorFlight Log Analysis**

Blackbox Lab is an experimental browser-based RotorFlight log analysis tool. It turns raw CSV blackbox data into plain-English engineering summaries.

## Current Features

- RotorFlight CSV import
- Header-row detection
- Channel mapping by name
- Flight-phase filtering
- Governor analysis
- Battery/power analysis
- ESC efficiency analysis
- Vincent plain-English verdict
- TXT engineering report export

## Project Philosophy

> The data doesn't lie.

The goal is not just to display telemetry, but to explain what the telemetry means.

## How to Use

1. Download or clone this repository.
2. Open `index.html` in a browser.
3. Load a RotorFlight CSV file.
4. Review the generated engineering report.
5. Export the TXT report if desired.
